# Frontend Mentor - Article Preview Component

A solution for the Article Preview challenge on [Frontend Mentor](https://www.frontendmentor.io) focused on reproducing the given layout, tooltip behavior, and responsive states.

## Screenshot

![Article Preview Component screenshot](./design/desktop-preview.jpg)

## Overview

- Recreated the desktop and mobile layouts pixel-close to the design specs.  
- Implemented the share tooltip with accessible toggle states.  
- Ensured the component adapts smoothly between 375px and 1440px viewports.

## Built With

- Semantic HTML5 markup  
- SCSS for modular styling  
- Vanilla JavaScript for the share interaction  
- Flexbox and CSS Grid for layout  
- Vite for local development
